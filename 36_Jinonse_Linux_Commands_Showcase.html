sort

The sort command sorts lines of text files. It supports alphabetic, numeric, and reverse sorting.

Usage:

sort filename.txt

awk

The awk command processes text files line by line and is used for pattern scanning and data extraction.

Usage:

awk '{print $1}' filename.txt

sed

The sed command is a stream editor used for text manipulation, such as finding and replacing text in a file.

Usage:

sed 's/old/new/g' filename.txt

cut

The cut command is used to extract specific sections of a file or text by column or character.

Usage:

cut -d',' -f2 filename.txt

tr

The tr command is used to translate or delete characters from text.

Usage:

echo "hello" | tr 'a-z' 'A-Z'

echo

The echo command is used to display text or variable values in the terminal.

Usage:

echo "Hello, World!"

export

The export command is used to set environment variables that are available to child processes.

Usage:

export VAR=value

history

The history command displays the list of previously executed commands.

Usage:

history

rsync

The rsync command is used to synchronize files and directories between systems.

Usage:

rsync -av source/ destination/

screen

The screen command is used to manage multiple terminal sessions within a single terminal window.

Usage:

screen

uniq

The uniq command removes duplicate lines from a sorted file.

Usage:

uniq filename.txt
